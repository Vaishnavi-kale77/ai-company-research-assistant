# AI Company Research Assistant

AI-powered company research tool built using Streamlit, OpenRouter, and Serper API.

## Features

- Company website detection
- Website crawling
- AI-powered analysis
- Competitor analysis
- Pain point analysis
- PDF report generation

## Tech Stack

- Streamlit
- OpenRouter
- Serper API
- BeautifulSoup
- ReportLab

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
